Welcome to My CSS Project

This page demonstrates a responsive layout built using CSS Flexbox. The header, main content area, and footer are stacked vertically, while the main content is split into two columns: the primary article and the sidebar.

Flexbox Layout Explained

The <body> element is a flex container with flex-direction: column. The <main> container is also a flex container, but with flex-direction: row, which is what puts the <article> and <aside> side-by-side.

The 2/3 and 1/3 Split

The <article> uses flex: 2, taking up two parts of the available space, while the <aside> uses flex: 1, taking up one part. This creates a perfect two-thirds/one-third column split, regardless of the screen size.
